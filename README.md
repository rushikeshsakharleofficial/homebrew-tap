# homebrew-tap

Homebrew tap for [rushikeshsakharleofficial](https://github.com/rushikeshsakharleofficial) tools.

## Usage

```bash
brew tap rushikeshsakharleofficial/tap
brew install code-outline-graph-go
```

## Tools

| Formula | Description |
|---------|-------------|
| `code-outline-graph-go` | Go code indexing and search MCP server |
